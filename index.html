Portofolio

Berpatisipasi dalam proyek Proyek Berbasis Langsung (PBL) pada semester 2, pengembangan sebuah website untuk penjualan barang atau jasa.
Menggunakan framework Laravel untuk pengembangan backend dan Tailwind CSS untuk desain frontend. Peran saya terutama berfokus pada pengembangan bagian backend dari aplikasi ini.

Proyek pengerjaan branding produk di SMK melibatkan pembuatan identitas unik untuk produk tertentu menggunakan Adobe Ilustrator dan Adobe Photoshop.
Ini meliputi penentuan nama produk, desain logo, pemilihan warna, serta strategi pemasaran yang sesuai dengan target pasar.
